# TwentyAndFiveEyeRest
A program that follows the correct procedure of resting for five minutes after every twenty minutes of computer use.


Option-1 scaffold: a cross-platform eye-rest reminder (Qt6 frontend, decoupled backend).

Quick start (Ubuntu/Debian):
1. `sudo ./install.sh`
2. `./run.sh` -> choose Build, then Run program.

This scaffold uses Qt6 (shared), Boost.Test, and CMake. The `run.sh` preserves a versioned working executable on successful build into `release_versions/v<semver>`.
